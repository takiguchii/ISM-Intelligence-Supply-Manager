# ISM - Intelligence Supply Manager

Foundation repository for the ISM vertical SaaS platform for restaurants.

## Sobre o Projeto

O ISM (Intelligence Supply Manager) é um ecossistema vertical SaaS projetado para a gestão inteligente de suprimentos no setor gastronômico. Este repositório contém a fundação técnica do projeto, utilizando uma arquitetura robusta e escalável.
